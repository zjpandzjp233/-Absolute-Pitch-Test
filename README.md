# Absolute-Pitch-Test

// Absolute Pitch Test: Sound Playback and Note Name Response Trainer // This module plays a sound, and the user responds with the name of the note heard.

主要功能：在88键或者在一个八度内为你随机弹奏一下音符，而你只需要输入刚刚弹奏了的按键的音名，拿下满分。
